# 💎 Simple 3D Parallax Card

A visually engaging 3D parallax card that creates a realistic depth effect using layered elements and mouse interaction. This project demonstrates advanced UI techniques to simulate real 3D movement on the web.

💎 Part of my 50+ Projects Challenge

---

## ✨ Features
- 🎯 Real 3D depth using translateZ layers
- 🖱️ Interactive mouse movement effect
- 🎨 Smooth animations and transitions
- 🌌 Modern dark UI design
- 📱 Responsive layout

---

## 🛠️ Built With
- HTML5
- CSS3 (3D Transforms, Flexbox)
- JavaScript (Mouse Events)
